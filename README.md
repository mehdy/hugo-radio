# Hugo Radio

This is a hugo theme for podcast usecases.

## Configuration

```toml
baseURL = "https://example.com/"
languageCode = "fa-ir"
title = "Radio Ravand"
theme = "radio"
googleAnalytics = "UA-12345-1"

[params]
    langDir = "rtl"
    author = "Mehdy Khoshnoody"
    contact = "info@example.com"
```
